# 🏥 Schedula – Doctor Appointment Scheduling System

A backend appointment scheduling platform built using NestJS, PostgreSQL, TypeORM and JWT Authentication.

https://schedula-asrith.onrender.com

# 🛠 Tech Stack

* NestJS
* TypeScript
* PostgreSQL
* TypeORM
* JWT Authentication
* Render

---

# 📂 Project Setup

## Clone Repository

```bash
git clone https://github.com/Asrithchowdary/schedula-asrith.git
```

## Install Dependencies

```bash
npm install
```

## Create Environment Variables

Create a `.env` file:

```env
DATABASE_URL=your_database_url
JWT_SECRET=your_secret
PORT=3000
```

## Run Application

```bash
npm run start:dev
```
